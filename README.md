# Eternal Grind

A task management Flutter application with calendar integration and recurring tasks.

## Features

- **Task Duration Options**: Once, daily, weekly, and custom recurring tasks
- **Calendar Integration**: Visual calendar with task indicators
- **Date-Based Filtering**: View tasks for any selected date
- **Real-Time Sync**: Firebase integration across devices
- **Modern UI**: Dark theme with responsive design

## Quick Start

```bash
git clone https://github.com/Soham-Patel1604/Eternal_Grind.git
cd Eternal_Grind
flutter pub get
flutter run
```

## Usage

1. **Add Tasks**: Click the floating action button
2. **Set Duration**: Choose once, daily, weekly, or custom
3. **Navigate Dates**: Use calendar or date picker
4. **Complete Tasks**: Check off tasks as you finish them

## Task Durations

- **Once**: Single occurrence on specific date
- **Daily**: Repeats every day from creation
- **Weekly**: Repeats on same weekday each week
- **Custom**: Daily tasks within date range
